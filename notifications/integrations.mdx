## Available Integrations

StatusPage.one supports multiple notification channels to ensure you never miss an alert.

## Email Notifications

Send alerts to email addresses.

**Setup:**
1. Navigate to Alerts
2. Add new integration
3. Select Email
4. Enter email address
5. Save

**Best For:** Team notifications, ticketing systems, documentation

## SMS Notifications

Receive text messages when monitors go down.

**Setup:**
1. Add new integration
2. Select SMS
3. Enter phone number with country code (e.g., +1-555-0123)
4. Save

**Best For:** Critical alerts, on-call engineers, immediate notifications

## Slack Integration

Post alerts to Slack channels.

**Setup:**
1. Create incoming webhook in Slack
2. Add new integration in StatusPage.one
3. Select Slack
4. Paste webhook URL
5. Save

**Best For:** Team collaboration, incident response, real-time awareness

## Webhook Integration

Send HTTP requests to custom endpoints.

**Setup:**
1. Add new integration
2. Select Webhook
3. Enter URL and configure headers
4. Save

**Best For:** Custom integrations, automation, third-party tools

## Microsoft Teams

Post to Teams channels.

**Setup:** Similar to Slack, using Teams incoming webhook

## Discord

Send notifications to Discord servers.

**Setup:** Create webhook in Discord, add to StatusPage.one

## Telegram

Send messages to Telegram chats.

**Setup:** Configure bot token and chat ID
